1. What is string slicing?

Answer:

String slicing is used to extract a part (substring) of a string.

Example:

name = "Python"
print(name[0:3])

Output:

Pyt

⸻

2. What is the syntax of string slicing?

Answer:

The syntax is:

string[start : stop : step]

* start → Starting index (inclusive)
* stop → Ending index (exclusive)
* step → Number of characters to skip

⸻

3. Is the stop index included in slicing?

Answer:

No.

The stop index is excluded.

Example:

text = "Python"
print(text[0:3])

Output:

Pyt

Characters at indices 0, 1, and 2 are returned. Index 3 is not included.

⸻

4. What is the default value of the start index?

Answer:

The default start index is 0.

Example:

text = "Python"
print(text[:3])

Output:

Pyt

⸻

5. What is the default value of the stop index?

Answer:

The default stop index is the length of the string.

Example:

text = "Python"
print(text[2:])

Output:

thon

⸻

6. What is the default value of the step?

Answer:

The default value of the step is 1.

String Properties

13. What is immutability?

Answer:

Immutability means an object cannot be changed after it is created.

Strings in Python are immutable.

⸻

14. Why are Python strings immutable?

Answer:

Strings are immutable because it improves:

* Memory efficiency
* Performance
* Security
* Reusability

⸻

15. Can you modify a character in a string?

Example:

name = "Python"
name[0] = "J"

Answer:

No.

Python raises:

TypeError: 'str' object does not support item assignment

⸻

16. How can you modify a string?

Answer:

Since strings are immutable, create a new string.

Example:

name = "Python"
name = "J" + name[1:]

Output:

Jython

Concatenation

17. What is string concatenation?

Answer:

Concatenation means combining two or more strings using the + operator.

Example:

first = "Hello"
second = "World"
print(first + second)

Output:

HelloWorld

⸻

18. Can you concatenate a string and an integer?

Answer:

No.

Example:

"Age: " + 25

Produces:

TypeError

Convert the integer first:

"Age: " + str(25)

⸻

19. What is the output?

Python Programming
