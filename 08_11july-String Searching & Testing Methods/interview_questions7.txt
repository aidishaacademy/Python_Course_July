1. What are string methods in Python?

Answer:

String methods are built-in functions associated with string objects that allow us to perform operations such as validation, searching, case conversion, and formatting.

Example:

text = "python"
print(text.upper())

⸻

2. Why are string methods useful?

Answer:

String methods help perform common text operations easily without writing complex logic. They improve code readability and reduce development time.

⸻

3. How do you call a string method?

Answer:

Use the dot (.) operator.

Syntax:

string_object.method_name()

Example:

name = "python"
print(name.upper())

⸻

4. Do string methods modify the original string?

Answer:

Most string methods return a new string because strings are immutable in Python.

Example:

text = "Python"
new_text = text.lower()

The original text remains unchanged.

⸻

isalpha()

5. What does isalpha() do?

Answer:

isalpha() returns True if all characters in the string are alphabetic letters.

Otherwise, it returns False.

⸻

6. What is the output?

"Python".isalpha()

Answer:

True

⸻

7. What is the output?

"Python123".isalpha()

Answer:

False

Because the string contains digits.

⸻

8. Does isalpha() allow spaces?

Answer:

No.

Example:

"Hello World".isalpha()

Output:

False

Spaces are not alphabetic characters.

⸻

isdigit()

9. What does isdigit() do?

Answer:

It returns True if every character in the string is a digit.

⸻

10. What is the output?

"12345".isdigit()

Answer:

True

⸻

11. What is the output?

"123abc".isdigit()

Answer:

⸻

12. Can isdigit() identify decimal numbers?

Example:

"12.5".isdigit()

Answer:

No.

Output:

False

The decimal point (.) is not considered a digit.

find()

29. What does the find() method do?

Answer:

It returns the index of the first occurrence of a substring.

If the substring is not found, it returns -1.

⸻

30. What is the output?

text = "Python Programming"
print(text.find("Python"))

Answer:

0

⸻

31. What is the output?

text = "Python Programming"
print(text.find("Java"))

Answer:

-1

⸻

32. What is the difference between find() and index()?

Answer:

find()	index()
Returns -1 if substring is not found	Raises ValueError if substring is not found

⸻

33. What is the output?

text = "banana"
print(text.find("na"))

Answer:

2

It returns the first occurrence.

Advanced Interview Questions

46. Why does find() return -1 instead of raising an exception?

Answer:

Returning -1 allows developers to safely check whether a substring exists without handling exceptions, making it suitable for search operations.

⸻

47. Why are methods like isalpha() and isdigit() useful in input validation?

Answer:

They simplify validation by providing built-in checks for common input patterns, reducing the need for custom logic and making code more reliable.

⸻

48. Which method would you use to validate an email address?

Answer:

There is no single string method to fully validate an email address. Basic checks can use methods like find() or endswith(), but robust validation is typically done using regular expressions (re module).

⸻

49. Can isalpha() and isalnum() recognize Unicode characters?

Answer:

Yes. They support Unicode letters and digits. For example, alphabetic characters from many languages are considered valid by isalpha().
