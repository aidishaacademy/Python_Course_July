1. What is a string in Python?

Answer:

A string is a sequence of Unicode characters enclosed within single quotes (' '), double quotes (" "), or triple quotes (''' ''' or """ """).

Example:

name = "Python"

⸻

2. Is a string a primitive data type in Python?

Answer:

Yes.

str is one of Python’s built-in basic data types used to store text.

⸻

3. What are Unicode characters?

Answer:

Unicode is a universal character encoding standard that represents letters, numbers, symbols, emojis, and characters from almost every language.

Example:

name = "भारत"
emoji = "😊"

Python strings support Unicode by default.

⸻

4. Why does Python use Unicode?

Answer:

Unicode allows Python programs to work with multiple languages and special characters, making applications international and portable.

⸻

5. How can you create a string in Python?

Answer:

Strings can be created using:

* Single quotes
* Double quotes
* Triple single quotes
* Triple double quotes

Example:

a = 'Python'
b = "Python"
c = '''Python'''
d = """Python"""

⸻

6. What is the difference between single and double quotes?

Answer:

There is no functional difference.

Both create string objects.

Example:

name1 = 'Python'
name2 = "Python"

⸻

7. When should we use triple quotes?

Answer:

Triple quotes are mainly used for:

* Multi-line strings
* Documentation (Docstrings)

Example:

message = """

to
Python
"""

⸻

8. What is a docstring?

Answer:

A docstring is a multi-line string used to document modules, classes, and functions.

Example:

def add():
    """Returns sum of two numbers"""

⸻

9. Is a string mutable?

Answer:

No.

Strings are immutable, meaning they cannot be modified after creation.

⸻

10. What does immutable mean?

Answer:

Immutable means the object cannot be changed after it is created.

If modifications are needed, Python creates a new string.

⸻

11. How do you access characters in a string?

Answer:

Characters are accessed using indexing.

Example:

name = "Python"
print(name[0])

Output:

P

⸻

12. What is indexing?

Answer:

Indexing is the process of accessing individual characters of a string using their position.

⸻

13. From where does positive indexing start?

Answer:

Positive indexing starts from 0.

Example:

name = "Python"
P  y  t  h  o  n
0  1  2  3  4  5

⸻

14. From where does negative indexing start?

Answer:

Negative indexing starts from -1, which represents the last character.

Example:

name = "Python"
P  y  t  h  o  n
-6 -5 -4 -3 -2 -1

⸻

15. Why do we use negative indexing?

Answer:

Negative indexing allows us to access characters from the end of the string without knowing its length.

44. Print all characters using indexing.

Answer:

    print(name[i])
