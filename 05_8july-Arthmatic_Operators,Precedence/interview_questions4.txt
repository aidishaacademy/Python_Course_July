1. What are arithmetic operators in Python?

Answer:

Arithmetic operators are symbols used to perform mathematical calculations on numbers.

Python supports 7 arithmetic operators:

* + Addition
* - Subtraction
* * Multiplication
* / Division
* // Floor Division
* % Modulus
* ** Exponentiation

⸻

2. Which arithmetic operators are available in Python?

Answer:

Python provides the following arithmetic operators:

+
-
*
/
/ /
%
**

Each operator performs a different mathematical operation.

3. What does the addition (+) operator do?

Answer:

The + operator adds two numbers.

Example:

a = 10
b = 20
print(a + b)

Output:

30

⸻

4. What does the subtraction (-) operator do?

Answer:

The - operator subtracts one value from another.

Example:

20 - 5

Output:

15

⸻

5. What does the multiplication (*) operator do?

Answer:

The * operator multiplies two values.

Example:

5 * 4

Output:

20

⸻

6. What is the division (/) operator?

Answer:

The / operator performs normal division and always returns a float.

Example:

10 / 2

Output:

5.0

⸻

7. Why does division always return a float?

Answer:

In Python 3, the / operator always returns a floating-point value, even if the result is a whole number.

Example:

8 / 4

Output:

2.0

⸻

8. What is floor division (//)?

Answer:

Floor division returns the largest whole number less than or equal to the result of the division.

Example:

10 // 3

Output:

3

⸻

Answer:

/	//

Normal division	Floor division

10. What is the modulus (%) operator?

Answer:
